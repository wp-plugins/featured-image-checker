Check if Featured Image has been set or not for all Post Types.

== Description ==
This plugin checks if the post has a featured image set or not. It displays “yes/no” on the post page on the backend.

== Installation ==

Installing "Starter Plugin" can be done either by searching for “Featured Image Checker” via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org.
2. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does this plugin have a settings page? =

No. This plugin has no settings page. If activated, this plugin shows a column “Featured Image” on the Post list page on the backend with either Yes or No depending on whether featured image has been set for the post or not.
